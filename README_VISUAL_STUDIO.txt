BELLA24 - varianta pentru Visual Studio

Cum o deschizi:
1. Dezarhivează folderul.
2. Intră în folderul Bella24_VisualStudio.
3. Dublu click pe Bella24.sln.
4. În Visual Studio, verifică să ai workload-ul/extensia Python instalată.
5. Dacă Visual Studio cere interpreter, alege Python 3.12 sau creează un virtual environment.
6. În terminalul Visual Studio rulează:

   pip install -r requirements.txt

7. Pornește proiectul cu butonul Start sau rulează:

   python app.py

8. Deschide în browser:

   http://127.0.0.1:5000

Variantă rapidă fără configurare Visual Studio:
- dublu click pe START_BELLA24_LOCAL.bat
- scriptul creează .venv, instalează requirements și pornește site-ul local.
